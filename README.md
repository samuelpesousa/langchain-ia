# LangGraph Streaming POC

Interface de chat inteligente com suporte a streaming em tempo real, utilizando **LangGraph** no backend e **React** no frontend.

## Tecnologias
- **Backend:** Python 3.12, LangGraph SDK, LangChain.
- **Frontend:** React (Vite), TypeScript, Tailwind CSS.
